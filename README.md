# TunePlay - Android Music Player App

TunePlay is a premium Spotify-inspired Android music player built using Kotlin and Jetpack Compose.

## Features
- Premium Spotify-style modern UI
- Synced mini player + main player
- Shuffle mode
- Favorite songs support
- Loop mode
- Splash screen animation
- Premium curved album artwork
- Shared MusicManager architecture
- Smooth music playback controls
- Beautiful dark premium theme
